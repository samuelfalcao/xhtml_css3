<h1> CRIANDO UMA PÁGINA HTML </h1>

<h2>Conteúdo do Exercício:</h2>

<strong>Desenvolva uma página HTML com o nome de índex.html, ao ser executada com o navegador deverá ter um topo, uma sidebar do lado direito do site com pelo menos uma imagem, por fim um rodapé simples.</strong>
<strong>No conteúdo desta página escreva um testo simples com 10 parágrafos e um título, utilize CSS para definir as dimensões e cada parte do site utilize cores de fundo para ficar mais fácil a identificação.</strong>
<strong>Lembre-se de usar as tags corretas para cada parte do site.</strong>
